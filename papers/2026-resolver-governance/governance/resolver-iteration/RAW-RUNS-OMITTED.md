# Raw per-run outputs (excluded)

The 4574 raw per-run decision transcripts (`runs/`) are bulk intermediate
data and are omitted from the public archive. The blind codings
(`coding-blind-map*.json`), per-round results (`*-results.md`), analysis
(`analysis/`), and code (`scripts/`) reproduce every reported number.
Raw run transcripts are available from the author on request.
